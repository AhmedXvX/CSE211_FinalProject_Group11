## Project Instructions

### Project Overview
EventsX is a comprehensive web application designed to simplify event planning and management. It allows users to browse upcoming events, register for them, and calculate event budgets using an intuitive interface. The project demonstrates responsive web design, dynamic content loading via PHP, and interactive JavaScript features.

### Installation & Setup
1. **Prerequisites:**
   - Ensure you have a local web server environment installed (e.g., XAMPP, WAMP, MAMP) that supports PHP and MySQL.

2. **Database Setup:**
   - Access your database management tool (e.g., phpMyAdmin).
   - Create a new database named `eventsx`.
   - Import the provided SQL script.

3. **Deploying the Project:**
   - Copy the entire project folder `CSE211_FinalProject_Group11` into your server's root directory (`htdocs` for XAMPP, `www` for WAMP).
   - Ensure the directory structure is preserved.

4. **Running the Application:**
   - Start your Apache and MySQL servers.
   - Open your web browser.
   - Navigate to `http://localhost/CSE211_FinalProject_Group11/Part2_Website_Project/pages/index.html` (adjust the path if you renamed the folder).
